# TESC Check-In

## Requirements
0. Node.js Version >= 8.2.1

## Installation
0. Clone Repository
1. Navigate to directory in bash
2. Run ```npm install```
3. Copy ```.env.example``` to a new file ```.env```
4. Enter all of the information into the ```.env``` file

## Development
1. Run ```npm start```
2. Navigate to ```http://localhost:8000/```
